Descripción detallada

Clase MapaDistancias , que ofrece funcionalidad en el manejo de una matriz de distancias entre varias ciudades.

Documentación de los datos miembro

string MapaDistancias::ciudades[NUMCIUDADES] [private]

Array que almacena los nombres de las distintas ciudades.

Tiene un tamaño fijo dado por la constante públlica NUMCIUDADES. Los índices de cada ciudad corrresponden con los índices de matriz. De esta forma, dentro de la clase se trabaja solo con índices mientras que en el main se trabaja con los nombres de las ciudades que devolerán los métodos recurrriendo a este dato miembro.

double MapaDistancias::matriz[NUMCIUDADES][NUMCIUDADES] [private]

Matriz que almacena las distancias entre las distintas ciudades con las que se trabajará.

Tiene un tamaño fijo dado por la constante públlica NUMCIUDADES. Cada índice corresponde a una ciudad de manera que matriz[i][j] es la distancia entre la ciudad i y la j.
